# Blog_API
> AJu Nuxt Blog XML API

## Build Setup

``` bash
# install dependencies
$ npm run install

# serve with hot reload at localhost:3001
$ npm run start
```

For Nuxt Blog Simple XML API

## config.json
> Must setting server config
``` json
{
	"server":{
		"db_url":""
	}
}
```
| Key1 | Key2 | 의미 |
| :-------- |:--------- | :-------- |
| `server` 	| `db_url` 	| 접근할 DB URL |
